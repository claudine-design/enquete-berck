Appartements Berck-sur-Mer

📋 Procédure de ménage — à lire avant de commencer

📥 Avant ménage

Matériel à emporter

  • Sac ménage + sac pressing + pack de linge (si besoin)
  • Panier de linge sale depuis le local ménage (si machine à faire)

À l'arrivée dans l'appartement

  • Vider son sac de ménage à un endroit (table, canapé, chaise)
  • Ramasser le linge sale des précédents locataires :
    • Tapis, torchon, essuie-mains, éponge → sac ménage
    • Linge de lit → sac pressing
  • Ouvrir le placard ménage
  • 📷 Prendre en photo le questionnaire rempli

Sortir du placard ménage

  • 🛁 SDB : produit WC, anti-calcaire, javel, sac poubelle
  • 🍳 Cuisine : café, thé, sucre, pastille lave-vaisselle, sac poubelle
  • 🛋️ Salon : questionnaire (si besoin), produit vitres, produit magique (vinaigre)

🧽 Ménage — avant de commencer

  • Lancer la machine à laver si besoin (linge sale du local ménage)
  • Mettre l'anti-calcaire sur toute la robinetterie (SDB + cuisine) + pare-douche ou baignoire
  • Mettre la javel sur les joints de la salle de bain
  • Produit WC dans les toilettes
  • Radiateurs : 12° l'hiver, éteints l'été

🛏️ Chambre

  • Vérifier l'alèse
  • Refaire le lit + déco
  • Déposer le pack de linge si besoin
  • Vérifier sous le lit
  • Poussière des meubles
  • Ouvrir la table de nuit
  • Faire les vitres si nécessaire

🛁 Salle de bain

  • Miroir
  • Plan de travail
  • Lavabo et robinet
  • Vérifier savon pour les mains + coton / coton-tige
  • Mettre en plus le tapis
  • Vérifier poubelle + remettre un sac propre si besoin
  • Rincer la douche
  • Vérifier le gel douche
  • Nettoyer paroi / robinetterie / bac de douche
  • Nettoyer la bonde de douche et enlever les cheveux

🚽 Toilette

  • Vérifier papier toilette (2 rouleaux)
  • Nettoyer les WC + tirer la chasse

🍳 Cuisine

  • Ouvrir tous les placards (vérifier qu'il n'y a pas de nourriture oubliée)
  • Nettoyer l'électroménager :
    • Micro-onde : intérieur + plaque tournante
    • Lave-vaisselle : ouvrir, allumer, vérifier liquide de rinçage (☀️) + sel (flèches)
    • Four : ouvrir + vérifier la plaque
    • Bouilloire : vider l'eau, détartrer au vinaigre si calcaire, rincer
    • Senseo : pas de capsule, nettoyer porteurs + récipient
  • Nettoyer le plan de travail
  • Évier + robinet + vérifier évacuation (buse en place)

🎁 Panier d'accueil

  • Dépliant info
  • Liquide vaisselle
  • Torchon, essuie-main, éponge
  • Pastille lave-vaisselle
  • Café / thé / sucre — selon le nombre de couchages (ex : 4 pers = 4 café / 4 thé / 4 sucres)
  • Gâteaux / chocolat
  • Vérifier la poubelle (nettoyer l'intérieur si besoin) + sac poubelle
  • Faire les vitres si nécessaire

🍽️ Salle à manger

  • Nettoyer table et chaises
  • Nettoyer les surfaces utiles, plan de travail et table

🛋️ Salon

  • Vérifier l'alèse du canapé
  • Refaire la déco du canapé
  • Poussière des meubles + table basse
  • Sur la table basse : livret d'accueil, questionnaire, télécommande
  • Faire les vitres si nécessaire

🌳 Jardin / Balcon

  • Vérifier et remettre en place le mobilier
  • Vérifier le brasero
  • Pas de mégots ni de crottes de chien

🧹 Aspirateur — partout

  • Sous les lits
  • Déplacer tables de nuit / portants papier toilette et brosse WC
  • Sous les meubles
  • Derrière les portes

🔍 Vérifications si besoin

  • Tête de lit
  • Plinthes
  • Poussière sur le radiateur

🚪 Avant de partir de l'appartement

  • Mettre le spray anti-odeur
  • Vérification finale de l'appartement
  • Refermer le placard ménage

🏁 Fin du ménage (retour au local)

  • Accrocher la machine (si elle a tourné)
  • Refaire le sac ménage :
    • 1 tapis
    • 1 torchon
    • 1 essuie-main
    • 3 lavettes
    • 2 rouleaux de papier toilette
    • 1 paquet de gâteaux / chocolat
💡 Si un produit est fini ou manquant dans le logement, le noter pour la personne qui fera le prochain ménage.
